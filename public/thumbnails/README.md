# Thumbnails de Streams

Esta carpeta contiene las imágenes en miniatura que aparecen en las tarjetas del carrusel de streams.

## Imágenes necesarias:

### Por categoría de stream:
- `conversando.jpg` - Para streams de "Just Chatting" / Conversando
- `minecraft.jpg` - Para streams de Minecraft
- `fortnite.jpg` - Para streams de Fortnite
- `gtav.jpg` - Para streams de GTA V
- `valorant.jpg` - Para streams de Valorant
- `league.jpg` - Para streams de League of Legends
- `variety.jpg` - Para streams de variedad
- `gaming.jpg` - Para streams generales de gaming

## Especificaciones técnicas:
- **Formato:** JPG o PNG
- **Tamaño recomendado:** 400x225px (16:9 ratio)
- **Peso:** Máximo 200KB por imagen
- **Calidad:** Alta resolución para verse bien en el carrusel

## Notas:
- Estas imágenes aparecen en el fondo de las tarjetas del carrusel
- Representan el tipo de contenido que está transmitiendo el streamer
- Se pueden reutilizar para diferentes streamers con la misma categoría
