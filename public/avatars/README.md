## Especificaciones técnicas:
- **Formato:** JPG o PNG
- **Tamaño recomendado:** 200x200px (mínimo)
- **Relación de aspecto:** 1:1 (cuadrada)
- **Peso:** Máximo 100KB por imagen
- **Calidad:** Buena resolución, imagen clara del rostro

## Notas:
- Las imágenes se mostrarán como círculos en el carrusel
- Asegúrate de que el rostro esté centrado
- Evita imágenes con mucho texto o elementos que se corten al convertir a círculo
